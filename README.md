### Things I've built

**[Emergency Network Web App](https://github.com/eemanazim/emergency_network_helping-web)**
A full-stack platform for asking or offering help during emergencies — PHP + MySQL backend, with a frontend designed to be fast and clear when it matters most.

**[CryptoBench](https://github.com/eemanazim/cryptobench)**
A Python tool for encrypting, decrypting, and benchmarking classical ciphers (Caesar, Vigenère, substitution), including frequency-analysis attacks against them without a known key.

**Student Task Manager App**
A React app for organizing academic tasks — priority tags, deadlines, completion tracking. Built around component architecture and hooks.

**Library Management System**
A PHP + MySQL system for tracking books, members, and issue/return records, built on a normalized relational schema.
